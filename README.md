# DESAFIO TÉCNICO
# book-publisher
## instruçoes:

#### dependencias:
ruby `-v 3.2.2`
node `-v 18.16.1`
npm `-v 9.5.1`

#### executar os seguintes comandos:
```console
% git clone https://github.com/annagodoy/book-publisher.git
% gem install bundler
% cd book-publisher
% bundle install
% rake db:setup
% npm install
% rails server
```

Após os comandos acima a aplicação deverá estar rodando em ``http://localhost:3000/``
